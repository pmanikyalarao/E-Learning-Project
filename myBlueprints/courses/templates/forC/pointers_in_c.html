Pointers in C

Pointers are a powerful feature in C programming that allow you to work directly with memory addresses. Understanding pointers is crucial for efficient memory management and for creating complex data structures like linked lists, trees, and more.

What is a Pointer?

A pointer is a variable that stores the memory address of another variable. Pointers provide a way to directly access and manipulate memory, enabling more efficient and flexible programming.

Declaring Pointers

To declare a pointer, use the * operator before the pointer's name. 

int *ptr; // Pointer to an integer

Initializing Pointers

Pointers should be initialized before use. You can assign a pointer the address of a variable using the & operator. 

int var = 10;
int *ptr = &var; // ptr now holds the address of var

Accessing Values through Pointers

You can access the value stored at the address pointed to by a pointer using the * operator (dereferencing). 

int value = *ptr; // value is now 10

Pointer Arithmetic

Pointers support arithmetic operations, allowing you to navigate through arrays and other data structures. 

int arr[] = {1, 2, 3, 4};
int *ptr = arr; // Points to the first element
int second = *(ptr + 1); // second is now 2

Common Uses of Pointers

  • Dynamic memory allocation
  • Passing large structures or arrays to functions without copying
  • Building complex data structures (e.g., linked lists, trees)

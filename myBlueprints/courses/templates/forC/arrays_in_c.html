Arrays in C

Arrays are a fundamental data structure in C that allow you to store multiple values of the same type in a single variable. They provide a convenient way to manage collections of related data and are widely used in programming.

What is an Array?

An array is a collection of elements, each identified by an index or key. The elements in an array are stored in contiguous memory locations, making it easy to access them using their indices.

Declaring and Initializing Arrays

You can declare an array by specifying its type and size. Arrays can be initialized at the time of declaration or assigned values later. 

int arr[5]; // Declaration of an array of integers with 5 elements
int arr2[] = {1, 2, 3, 4, 5}; // Declaration and initialization

Accessing Array Elements

Elements in an array are accessed using their index, which starts from 0. You can read or modify the values directly. 

int first = arr2[0]; // Accessing the first element (1)
arr2[1] = 10; // Modifying the second element (now 10)

Multidimensional Arrays

C also supports multidimensional arrays, such as two-dimensional arrays, which can be used to represent matrices or tables. 

int matrix[3][3]; // Declaration of a 3x3 matrix
matrix[0][1] = 5; // Accessing the element in the first row, second column

Common Uses of Arrays

  • Storing collections of data (e.g., lists, matrices)
  • Implementing algorithms that require batch processing of data
  • Efficiently passing large amounts of data to functions

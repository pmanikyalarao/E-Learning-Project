Strings in C

Strings in C are arrays of characters that are used to store and manipulate text. Unlike many other programming languages, C does not have a dedicated string type; instead, strings are represented as character arrays terminated by a null character (`\0`).

Declaring and Initializing Strings

You can declare a string in C by defining a character array. Strings can be initialized at the time of declaration or assigned later. 

char str1[6] = "Hello"; // Declaration with initialization
char str2[] = "World"; // Size is determined automatically

Accessing String Elements

Individual characters in a string can be accessed using indices, similar to arrays. 

char firstChar = str1[0]; // Accessing the first character ('H')

Common String Functions

The C Standard Library provides several functions for string manipulation, defined in the `` header. Some commonly used functions include:

  • strlen(str): Returns the length of the string.
  • strcpy(dest, src): Copies the source string to the destination.
  • strcat(dest, src): Appends the source string to the destination.
  • strcmp(str1, str2): Compares two strings for equality.

Example of String Manipulation

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    
    strcat(str1, str2); // str1 now contains "HelloWorld"
    printf("%s\n", str1); // Output: HelloWorld
    return 0;
}

Loop Control Statements in C

Loop control statements in C allow you to execute a block of code repeatedly, based on a specified condition. They are essential for performing repetitive tasks efficiently and are a fundamental concept in programming.

Types of Loop Control Statements

1. For Loop

The for loop is used when the number of iterations is known beforehand. It consists of three parts: initialization, condition, and increment/decrement. 

for (initialization; condition; increment) {
    // Code to execute in each iteration
}

2. While Loop

The while loop is used when the number of iterations is not known and depends on a condition being true. The code block is executed as long as the condition remains true. 

while (condition) {
    // Code to execute in each iteration
}

3. Do-While Loop

The do-while loop is similar to the while loop, but it guarantees that the code block is executed at least once before checking the condition. 

do {
    // Code to execute in each iteration
} while (condition);

Loop Control Statements

In addition to the loops themselves, C provides control statements to manipulate the flow within loops.

  • break: Terminates the loop immediately, skipping any remaining iterations.
  • continue: Skips the current iteration and moves to the next iteration of the loop.

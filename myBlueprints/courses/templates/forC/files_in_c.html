File Concepts in C

File handling in C allows you to read from and write to files, providing a way to persist data beyond the execution of a program. Understanding file concepts is essential for managing data efficiently and effectively.

File Types

In C, files can be classified into two main types:

  • Text Files: Store data in human-readable format, where each line of the file represents a record.
  • Binary Files: Store data in a format that is not human-readable, allowing for efficient storage of complex data types.

File Operations

C provides a set of functions to perform operations on files, defined in the stdio.h header. The basic operations include:

  • fopen(): Opens a file and returns a pointer to it.
  • fclose(): Closes an opened file.
  • fread(): Reads data from a file.
  • fwrite(): Writes data to a file.
  • fprintf(): Writes formatted output to a file.
  • fscanf(): Reads formatted input from a file.

Opening a File

The fopen function is used to open a file. You can specify the mode of the file, such as read (`"r"`), write (`"w"`), or append (`"a"`). 

FILE *file = fopen("example.txt", "r"); // Opens a file for reading
if (file == NULL) {
    // Handle error
}

Reading from a File

You can read data from a file using functions like fgets or fscanf. 

char buffer[100];
fgets(buffer, sizeof(buffer), file); // Reads a line from the file

Writing to a File

To write data to a file, use functions like fprintf or fputs. 

fprintf(file, "Hello, World!\n"); // Writes formatted output to the file

Closing a File

Always close a file after you are done with it to free resources using the fclose function. 

fclose(file); // Closes the file

Strings in Java

Strings in Java are objects that represent a sequence of characters. They are widely used in programming to handle text data. Java provides a rich set of methods for string manipulation, making it easy to perform various operations.

1. Creating Strings

In Java, you can create a string in two ways:

String Literal

String greeting = "Hello, World!"; // Using string literal

Using the String Constructor

String greeting = new String("Hello, World!"); // Using the String class constructor

2. Common String Methods

Java's String class provides many methods for manipulating strings. Here are some commonly used methods:

  • length(): Returns the length of the string.
    • int len = greeting.length();
  • charAt(int index): Returns the character at the specified index.
    • char firstChar = greeting.charAt(0); // 'H'
  • substring(int beginIndex, int endIndex): Extracts a substring from the string.
    • String sub = greeting.substring(0, 5); // "Hello"
  • toLowerCase(): Converts the string to lowercase.
    • String lower = greeting.toLowerCase(); // "hello, world!"
  • toUpperCase(): Converts the string to uppercase.
    • String upper = greeting.toUpperCase(); // "HELLO, WORLD!"
  • trim(): Removes leading and trailing whitespace.
    • String trimmed = "  Hello, World!  ".trim(); // "Hello, World!"

3. String Concatenation

You can concatenate strings using the + operator or the concat() method:

String name = "Alice";
String message = greeting + " My name is " + name; // Using + operator
String message = greeting.concat(" My name is ").concat(name); // Using concat()

4. String Comparison

To compare strings, use the equals() method for content comparison, and compareTo() for lexicographical comparison:

boolean isEqual = greeting.equals("Hello, World!"); // true
int comparison = greeting.compareTo("Hello, World!"); // 0 (equal)

5. String Immutability

Strings in Java are immutable, meaning once a string is created, it cannot be changed. Any modification results in the creation of a new string. For example:

String original = "Hello";
String modified = original.replace('H', 'J'); // Creates a new string "Jello"

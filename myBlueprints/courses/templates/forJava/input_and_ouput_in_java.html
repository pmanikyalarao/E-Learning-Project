Input and Output in Java

Input and output (I/O) operations are crucial in Java programming, enabling interaction with users, reading from files, and displaying data. Java provides various classes and methods for handling I/O operations efficiently.

1. Input in Java

Java provides several ways to accept user input, with the most common method being through the Scanner class. This class is part of the java.util package and allows easy reading of different data types.

Using the Scanner Class

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();
        System.out.println("Hello, " + name + "!");
        scanner.close();
    }
}

2. Output in Java

For displaying output, Java provides several options. The most commonly used methods are through the System.out class, which allows you to print data to the console.

Using System.out

System.out.println("Hello, World!"); // Prints a line with a newline
System.out.print("Hello, World!"); // Prints without a newline

Formatted Output

Java also allows formatted output using the printf method:

System.out.printf("Name: %s, Age: %d%n", name, age);

3. File Input and Output

In addition to console I/O, Java provides classes for reading from and writing to files. The FileReader and FileWriter classes are commonly used for text file operations.

Reading from a File

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReadExample {
    public static void main(String[] args) {
        try (BufferedReader br = new BufferedReader(new FileReader("input.txt"))) {
            String line;
            while ((line = br.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Writing to a File

import java.io.FileWriter;
import java.io.IOException;

public class FileWriteExample {
    public static void main(String[] args) {
        try (FileWriter writer = new FileWriter("output.txt")) {
            writer.write("Hello, World!");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Multidimensional Arrays in Java

Multidimensional arrays are arrays of arrays, allowing you to store data in multiple dimensions. The most common type is the two-dimensional array, which can be visualized as a table or matrix. Multidimensional arrays can also have more than two dimensions.

1. Declaring Multidimensional Arrays

In Java, you can declare a multidimensional array by specifying the data type followed by multiple sets of square brackets. Hereâ€™s how to declare a two-dimensional array:

int[][] matrix; // Declares a two-dimensional array

2. Initializing Multidimensional Arrays

You can initialize a multidimensional array at the time of declaration or later in the code. Here are two common methods:

Initialization at Declaration

int[][] matrix = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

Initialization After Declaration

int[][] matrix = new int[3][3]; // Creates a 3x3 matrix
matrix[0][0] = 1; // Assigns value to the first element
matrix[1][2] = 6; // Assigns value to the second row, third column

3. Accessing Elements in Multidimensional Arrays

You can access elements in a multidimensional array using multiple indices:

int value = matrix[1][2]; // Accesses the element in the second row, third column

4. Iterating Through Multidimensional Arrays

To iterate through the elements of a multidimensional array, you can use nested loops:

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println(); // Moves to the next line after each row
}

5. Higher-Dimensional Arrays

Java also supports higher-dimensional arrays. For example, a three-dimensional array can be declared and initialized as follows:

int[][][] threeDArray = {
        {
            {1, 2},
            {3, 4}
        },
        {
            {5, 6},
            {7, 8}
        }
    };

You can access elements in a three-dimensional array using three indices:

int value = threeDArray[1][0][1]; // Accesses the element at index [1][0][1]

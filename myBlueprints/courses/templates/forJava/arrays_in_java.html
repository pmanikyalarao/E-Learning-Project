Arrays in Java

Arrays are a fundamental data structure in Java that allows you to store multiple values of the same type in a single variable. They provide a way to group related data together, making it easier to manage and manipulate collections of items.

1. Creating Arrays

In Java, you can create an array by specifying the data type followed by square brackets. There are two common ways to declare and initialize an array:

Declaration and Initialization

int[] numbers = new int[5]; // Declares an array of integers with 5 elements
numbers[0] = 10; // Assigns a value to the first element

Array Literal

int[] numbers = {10, 20, 30, 40, 50}; // Declares and initializes an array

2. Accessing Array Elements

Array elements can be accessed using their index, which starts at 0. You can retrieve or modify the values in an array using this index:

int firstNumber = numbers[0]; // Accesses the first element
numbers[1] = 25; // Modifies the second element

3. Length of an Array

The length of an array can be obtained using the length property:

int length = numbers.length; // Gets the number of elements in the array

4. Iterating Through Arrays

You can use loops to iterate through the elements of an array. The for loop and the enhanced for loop (also known as the "for-each" loop) are commonly used:

Using a For Loop

for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Using a For-Each Loop

for (int number : numbers) {
    System.out.println(number);
}

5. Multi-Dimensional Arrays

Java also supports multi-dimensional arrays, which are arrays of arrays. The most common type is the two-dimensional array:

int[][] matrix = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

You can access elements in a multi-dimensional array using multiple indices:

int value = matrix[1][2]; // Accesses the element at row 1, column 2

User Input in Python

In Python, user input can be easily captured using the built-in input() function. This allows programs to interact with users by accepting data from the keyboard.

Using the input() Function

The input() function reads a line from the user input and returns it as a string. The basic syntax is: 

user_input = input("Enter your input: ")

Example

Here’s a simple example that asks the user for their name and prints a greeting: 

name = input("What is your name? ")
print("Hello, " + name + "!")

Type Conversion

Since input() returns a string, you may need to convert the input to other data types. For example, to get an integer: 

age = int(input("Enter your age: "))

Example with Type Conversion

Here’s how to ask for a number and perform calculations: 

num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
sum = num1 + num2
print("The sum is:", sum)

Handling Invalid Input

It’s important to handle potential errors when converting input. You can use try...except blocks to catch exceptions: 

try:
    number = int(input("Enter a number: "))
except ValueError:
    print("That's not a valid number!")

Conclusion

User input is a fundamental part of interactive Python programs. By utilizing the input() function along with type conversion and error handling, you can create responsive applications that engage users effectively.

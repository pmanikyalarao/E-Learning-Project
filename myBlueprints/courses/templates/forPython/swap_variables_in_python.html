Swapping Variables in Python

Swapping variables is a common task in programming where the values of two variables are exchanged. In Python, this can be done easily and efficiently using tuple unpacking.

Swapping Variables Using Tuple Unpacking

Python allows you to swap variables in a single line using tuple unpacking. The syntax is as follows: 

a, b = b, a

Example

Here’s a simple example demonstrating how to swap two variables: 

a = 5
b = 10
print("Before swap: a =", a, ", b =", b)

# Swapping
a, b = b, a

print("After swap: a =", a, ", b =", b)

Swapping Without a Temporary Variable

Unlike many other programming languages, Python does not require a temporary variable to swap values, making the process straightforward.

Example Without a Temporary Variable

Here’s another method using arithmetic operations (note that this approach can lead to overflow with large numbers): 

a = 5
b = 10

# Swapping using arithmetic operations
a = a + b
b = a - b
a = a - b

print("After swap: a =", a, ", b =", b)

Conclusion

Swapping variables in Python is simple and can be achieved in a clean and efficient way using tuple unpacking. This feature highlights Python's flexibility and readability, making code easier to write and understand.

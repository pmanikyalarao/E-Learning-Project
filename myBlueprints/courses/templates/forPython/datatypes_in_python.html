Data Types in Python

Python supports various data types, allowing you to store and manipulate different kinds of data. Understanding these data types is crucial for effective programming.

Built-in Data Types

  • Numeric Types:
    • int: Represents integers, e.g., 10, -5.
    • float: Represents floating-point numbers, e.g., 3.14, -0.001.
    • complex: Represents complex numbers, e.g., 2 + 3j.
  • Sequence Types:
    • str: Represents strings, e.g., "Hello".
    • list: Represents ordered, mutable collections, e.g., [1, 2, 3].
    • tuple: Represents ordered, immutable collections, e.g., (1, 2, 3).
  • Mapping Type:
    • dict: Represents key-value pairs, e.g., {"name": "Alice", "age": 25}.
  • Set Types:
    • set: Represents unordered collections of unique elements, e.g., {1, 2, 3}.
    • frozenset: Represents immutable sets.
  • Boolean Type:
    • bool: Represents truth values, True or False.

Type Checking and Conversion

You can check the type of a variable using the type() function:

value = 10
print(type(value))  # Output:

You can also convert between types using functions like int(), float(), and str():

num = 5.5
int_num = int(num)  # Converts float to int

Conclusion

Understanding data types in Python is fundamental to writing effective code. Each data type has its own characteristics and use cases, making it essential to choose the right type for your data.

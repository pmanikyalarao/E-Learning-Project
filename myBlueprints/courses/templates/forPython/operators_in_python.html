Operators in Python

Operators are special symbols that perform operations on variables and values. Python supports various types of operators that can be categorized as follows:

1. Arithmetic Operators

These operators are used to perform mathematical operations:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Floor Division: //
  • Modulus: %
  • Exponentiation: **
x = 10
y = 3
print(x + y)  # Output: 13
print(x // y)  # Output: 3

2. Comparison Operators

These operators compare two values and return a Boolean result:

  • Equal to: ==
  • Not equal to: !=
  • Greater than: >
  • Less than: <
  • Greater than or equal to: =
  • Less than or equal to: <=
a = 5
b = 10
print(a < b)  # Output: True

3. Logical Operators

These operators are used to combine conditional statements:

  • And: and
  • Or: or
  • Not: not
x = True
y = False
print(x and y)  # Output: False

4. Assignment Operators

These operators are used to assign values to variables:

  • Assign: =
  • Add and assign: +=
  • Subtract and assign: -=
  • Multiply and assign: *=
  • Divide and assign: /=
c = 5
c += 3  # c is now 8

5. Bitwise Operators

These operators perform operations on binary numbers:

  • AND: &
  • OR: |
  • XOR: ^
  • NOT: ~
  • Left Shift: <<
  • Right Shift: >>

Conclusion

Operators in Python are essential for performing calculations and comparisons. Understanding how to use them effectively is crucial for writing efficient code.

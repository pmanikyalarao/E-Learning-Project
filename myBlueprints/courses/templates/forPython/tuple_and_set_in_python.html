Tuples and Sets in Python

Tuples

A tuple is an immutable sequence type in Python. Tuples are used to store multiple items in a single variable. They can hold a collection of different data types, and their elements cannot be changed after creation.

Creating a Tuple

Tuples are created by placing a comma-separated sequence of items inside parentheses:

my_tuple = (1, 2, 3)

Accessing Tuple Elements

You can access elements in a tuple using indexing:

first_item = my_tuple[0]  # Accesses the first item (1)

Benefits of Tuples

  • Immutable: Ensures data integrity.
  • Faster than lists: Less memory overhead.
  • Can be used as dictionary keys.

Sets

A set is an unordered collection of unique items. Sets are mutable, meaning you can add or remove items after creation, but they do not allow duplicate elements.

Creating a Set

Sets are created by placing a comma-separated sequence of items inside curly braces or by using the set() constructor:

my_set = {1, 2, 3}
my_set = set([1, 2, 3])

Adding and Removing Elements

You can add elements using the add() method and remove them with the remove() method:

my_set.add(4)  # Adds 4 to the set
my_set.remove(2)  # Removes 2 from the set

Benefits of Sets

  • Unordered: Ideal for membership testing.
  • Unique elements: Automatically removes duplicates.
  • Supports set operations: Union, intersection, and difference.

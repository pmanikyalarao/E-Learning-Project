Lists in Python

Lists are a built-in data type in Python that allow you to store an ordered collection of items. They are mutable, meaning you can change their content after creation.

Creating a List

You can create a list by placing a comma-separated sequence of items inside square brackets:

fruits = ["apple", "banana", "cherry"]

Accessing List Elements

List elements can be accessed using their index, starting from 0:

print(fruits[0])  # Output: apple

Modifying Lists

Lists can be modified by adding, removing, or changing elements:

  • Append: Adds an element to the end of the list.
    • fruits.append("orange")
  • Insert: Inserts an element at a specified index.
    • fruits.insert(1, "kiwi")
  • Remove: Removes the first occurrence of a specified element.
    • fruits.remove("banana")
  • Pop: Removes and returns the last element (or the element at a specified index).
    • last_fruit = fruits.pop()

List Operations

Several operations can be performed on lists:

  • Length: Use len() to get the number of elements in a list.
    • print(len(fruits))  # Output: Number of items in the list
  • Sorting: Use sort() to sort the list in ascending order.
    • fruits.sort()
  • Reversing: Use reverse() to reverse the order of the list.
    • fruits.reverse()

Slicing Lists

You can access a range of elements using slicing:

sublist = fruits[1:3]  # Gets elements from index 1 to 2

Conclusion

Lists are a powerful and versatile data structure in Python. They allow you to store collections of items and provide numerous methods for manipulating that data effectively.
